Global Methods

Method Global:ClearPlayerEvents [-] [+]

Unbinds event handlers for either all Player events, or one type of Player event.

If event_type is nil, all Player event handlers are cleared.

Otherwise, only event handlers for event_type are cleared.

Synopsis

ClearPlayerEvents()

ClearPlayerEvents( event_type )

Arguments

number event_type

The event whose handlers will be cleared, see Global:RegisterPlayerEvent.

Valid numbers: integers from 0 to 4,294,967,295.

Returns

Nothing.
