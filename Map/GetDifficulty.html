Method Map:GetDifficulty [-] [+]

Returns the difficulty of the Map.

Always returns 0 if the expansion is pre-TBC.

Synopsis

Arguments

None.

Returns

number difficulty

Valid numbers: integers from -2,147,483,647 to 2,147,483,647.
