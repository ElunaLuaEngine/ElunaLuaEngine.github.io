Method WorldObject:GetNearestPlayer [-] [+]

Returns the nearest Player object in sight of the WorldObject or within the given range

Synopsis

Valid numbers: integers from 0 to 4,294,967,295.

Valid numbers: integers from 0 to 4,294,967,295.
